== Instrucciones ==

#if issubclass(type, SentenceCompletion)

El propósito de este ejercicio es ofrecerte una oportunidad de producir una
frase completa y comparar tu pronunciación con la original. Por lo tanto el
ejercicio es puramente aural-oral, sin tarea escrita.

Si deseas comprobar tus respuestas con el texto escrito, puedes usar la
transcripción de la grabación.  Al no tratarse aquí de un ejercicio escrito,
tus resultados no se pueden comprobar de forma automática.

#endif
#if issubclass(type, Reading)

Lee el texto e intenta comprender todo lo que puedas. Usa un diccionario si es
necesario. En muchos casos la información podrá serte útil para tus propios
propósitos, pero también se necesitará en los ejercicios posteriores.

#endif
#if issubclass(type, Listening)

Escucha la grabación para practicar tu comprensión de los hablantes nativos.

#endif

########## _ChoiceBasedExercise subclasses ##########

#if issubclass(type, MultipleChoiceQuestions)

Cada pregunta de este ejercicio viene seguida de dos o más respuestas
posibles. Sólo una respuesta es correcta.

#endif
#if issubclass(type, Selections)

Se ofrecen varios grupos de dos o tres frases o expresiones similares. Sólo una
opción en cada grupo es correcta.  Debes decidir cuál es la correcta.

#endif
#if issubclass(type, TrueFalseStatements)

Cada frase de este ejercicio viene seguida de dos controles llamados
'>>${TRUE}<<' y '>>${FALSE}<<' (Verdadero y Falso).  Decide si la frase es
verdadera o falsa y pulsa el botón correspondiente.

#endif
#if issubclass(type, GapFilling)

Elige la palabra correcta para rellenar un espacio en una frase.  Para cada
espacio se te ofrecen varias posibilidades. Sólo una es correcta.

#endif
#if issubclass(type, _ChoiceBasedExercise)

Inmediatamente después de elegir tu respuesta se te indica si es correcta o no.
Cuando te equivocas de contestación, puedes probar de nuevo hasta encontrar la
respuesta correcta.  Los resultados a continuación del ejercicio indican
cuántas repuestas acertaste al primer intento (ver la sección [#indicators]
abajo).

#endif

########## _FillInExercise subclasses ##########

#if issubclass(type, VocabExercise)

Hay dos maneras de hacer este ejercicio: oral y escrito.  Haz el ejercicio de
las dos maneras para sacar el máximo provecho.

Como ejercicio oral es muy fácil. Consulta la lista de vocabulario y decide
cuál es la traducción correcta de cada palabra o expresión.  A continuación del
cuadro de entrada de texto para cada elemento hay un botón >${Play}<<
(Reproducción), que te permite escuchar la respuesta correcta.  Repite la
respuesta para practicar la pronunciación correcta.  Algunos elementos tienen
más de una respuesta correcta, de modo que puede haber múltiples botones para
reproducir cada una.  Como no es posible comprobar tus respuestas orales, los
resultados no están disponibles. Si deseas ver tu puntuación, debes hacer el
ejercicio por escrito.

Para hacer el ejercicio por escrito, simplemente teclea la traducción de cada
elemento dentro de la casilla de entrada de texto.  Ten cuidado de usar
mayúsculas cuando proceda, ya que una palabra sin mayúscula cuando ésta es
necesaria siempre se considera incorrecta. Cuando se trata de una frase
completa, también debes usar la puntuación correcta.

#endif
#if issubclass(type, Transformation)

Tu objetivo es transformar una estructura determinada en una estructura
diferente, por ejemplo convirtiendo una frase afirmativa en pregunta.

#endif
#if issubclass(type, Substitution)

Utiliza la indicación para producir otra frase con la misma estructura.  A cada
frase le sigue un texto entre paréntesis.  Remplaza la parte correspondiente de
la frase usando este texto.

#endif
#if issubclass(type, _Cloze)

Tu objetivo en este ejercicio es rellenar los espacios en un texto más
largo. Sólo hay una respuesta correcta para cada espacio en blanco.

#endif
#if issubclass(type, Dictation)

Primero escucha el dictado completo una vez y a continuación desplázate al
cuadro grande de entrada de texto. Dentro del cuadro puedes usar teclas rápidas
para reproducir secciones de la grabación. Típicamente puedes reproducir una
sección usando la tecla *>* (signo 'más de') y teclear lo que
escuchas. Entonces vuelves a pulsar la tecla *>* de nuevo para reproducir la
siguiente sección y teclear ésta también, y así sucesivamente hasta que llegues
al final de la grabación.  Entonces puedes usar las teclas de evaluación (o los
botones de abajo) para evaluar el texto entero.

Presta una atención especial a puntuación y mayúsculas, ya que causan la
mayoría de los problemas al evaluar tu respuesta. Teclea todas las frases tal y
como las oyes, usando el carácter de puntuación apropiado y un solo espacio
después de cada frase. No se permite espacio después de la última frase.

/Nota:/ Naturalmente puedes reproducir la grabación las veces que quieras a fin
de comprenderla, aunque dos intentos deberían ser suficientes en condiciones
normales.

#else
#if issubclass(type, _FillInExercise)

Puedes comprobar cada respuesta individualmente usando las teclas rápidas (ver
la sección [#keys] para más información).  Cuando tu respuesta es evaluada como
incorrecta y no sabes por qué, comprueba siempre si has usado la puntuación
correcta y mayúsculas donde fueran necesarias. Estas dos cosas producen la
mayor confusión.

Usa el [#panel] en la parte inferior del ejercicio para evaluar todas las
respuestas a la vez.

Si tu navegador no soporta JavaScript, los elementos interactivos y la
evaluación automática no funcionan.  Puedes seguir usando los materiales, pero
tendrás que comprobar tus repuestas manualmente con las [#answer-sheets].

#endif
#endif

#if issubclass(type, _FillInExercise)

== Teclas Rápidas == keys

En todos los ejercicios en los que debes teclear texto dentro de un cuadro de
texto, puedes usar las dos teclas rápidas descritas a continuación.

Enter
   Utiliza esta tecla dentro del cuadro de texto para evaluar la respuesta
   actual. Escucharás una señal y en el caso de un error, el cursor se mueve a
   la posición del primer carácter incorrecto dentro del texto.  De esta manera
   puedes localizar el error, corregirlo y evaluar de nuevo.  Cuando no sepas
   corregir un error, puedes usar la tecla ``hint'' ('pista') descrita a
   continuación.

Ctrl-Space
   Esta función se llama una 'pista' (``hint'').  Te ayuda en el caso de que no
   sepas la respuesta o no sepas corregir un error en tu respuesta.
   Simplemente pulsa esta combinación de teclas (manteniendo pulsada la tecla
   Ctrl (control), pulsa la barra espaciadora) y una letra de la respuesta
   correcta se añade automáticamente. Si ya has tecleado algo de texto, el
   cursor se desplazará hasta después del último carácter correcto y se añade
   el siguiente. Esto significa que si hay algo de texto después del cursor,
   contiene al menos un error. Intenta localizar este error y corregirlo. Luego
   puedes evaluar tu respuesta usando la tecla 'Enter' (ver arriba) o usar
   ``pista'' de nuevo, hasta que encuentres la respuesta completa.

#if issubclass(type, Dictation)

También, en el cuadro de texto del ejercicio de dictado, puedes usar las
siguientes teclas para controlar la reproducción de la grabación:

> /(sign 'más de')/
   Reproducir siguiente sección de la grabación.

< /(signo 'menos de')/
   Reproducir sección previa de la grabación.

Ctrl-Enter
   Volver a reproducir la sección actual (última reproducida).

#endif
#endif

#if issubclass(type, _InteractiveExercise)

== Indicadores == indicators

El panel de indicadores situado debajo del ejercicio muestra los siguientes
valores:

#if issubclass(type, Dictation)

>>${Result:}<<
   Muestra el resultado de la última evaluación (puedes evaluar tu respuesta
   pulsando la tecla 'Enter' o mediante el botón >>${Evaluate}<< debajo del
   ejercicio).

#else

>>${Correct:}<<
   Muestra el número y el porcentaje de respuestas correctas. La primera pareja
   de números muestra los resultados de todas las respuestas actuales. Si no
   las contestaste todas correctamente al primer intento, también hay una
   segunda pareja de números que indican cuánta respuestas acertadas tuviste al
   primer intento. Usa el botón >>${Reset}<< para empezar otra vez.

>>${Answered:}<<
   Muestra el número de tareas que ya has completado.  Por ejemplo 4/10
   significa que has contestado cuatro de diez preguntas así que deberías
   terminar las seis que restan.

#endif

== Panel de Control == panel

El panel de control situado debajo del ejercicio contiene los botones
siguientes:

>>${Evaluate}<<
   Evaluar el ejercicio entero. Si se encuentra un error, el cursor se desplaza
   a la primera respuesta incorrecta. Dentro de un cuadro de texto, el cursor
   también se desplaza al primer carácter incorrecto de tu respuesta.

>>${Fill}<<
   Completar el ejercicio entero con las respuestas correctas.

>>${Reset}<<
   Borrar todas tus respuestas y empezar de nuevo.

== Hojas de Respuestas == answer-sheets

Las Hojas de Respuestas (>>${answer sheet}<<) contienen todas las respuestas
correctas de todos los ejercicios.  Puedes acceder a las Hojas de Respuesta
desde el Indice del Curso, o para más comodidad puedes usar el enlace del signo
de interrogación '?'  después de cada elemento del ejercicio. Este enlace te
lleva directamente a la respuesta correspondiente en la Hoja de Respuestas.
