= The Eurochance Course =

The Eurochance language course (see http://eurochance.brailcom.org) was the
initial implementation of a course using the LCG framework.  It is an example
of an implementation of a /Custom Interface/ on top of LCG API.  See [usage]
for more details.

This document should serve as a reference for creation of the input data for
the course.
